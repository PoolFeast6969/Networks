Networks
Types
  • Local Area Network (LAN)
    • LAN refers to a group of computers linked together across a small geographical area (ie. a building or a group of buildings close together)
    • For example, your school would have a LAN
    • LANs are usually connected with coaxial or CAT5 cables
  • Wide Area Network (WAN)
    • WAN refers to a group of LANs linked together across a large geographical area
    • For example, an organisation with many locations across the city, state or overseas
    • WANs are usually connected using ISDN landlines, satellite or internet
  • Peer to Peer (P2P)
    • P2P refers to computers communicating and sharing information directly with each other
    • This is not a very efficient way to set up the network and is only used in small situations
  • Client / Server
    • Client / Server refers to when there is a master computer (server) linked to a number of workstation (clients)
    • The server is usually more expensive and faster as it is the one which controls the sharing of information with the clients
    • Unlike P2P, the clients only communicate with the server, they do not communicate with each other
Hardware
  • Network Interface Card (NIC)
    • A NIC is used to link a computer or server to a network
    • NICs are also used in any other resource (eg printer) that is linked to a network
    • A NIC connects to the network by wired or wireless transmission media
  • Modem
    • A modem is a device that sends digital signals via a telephone line
    • When uploading messages, the modem modulates, which means that it converts the digital data into analogue signal so that it can be sent via the telephone line
    • When downloading messages, the modem demodulates, which means it converts the sound back into a digital signal so it can be understood by the computer
  • Cables
    • Cables are used to connect computers and devices to a network
    • By transferring electronic signals through strands of material that conduct electromagnetic waves
  • Switch
    • A switch is a connection point that links to a number of cables coming from various computers or devices. It stores address information about all of these devices
    • The switch receives the information from the device and determines the correct path for the message to be sent
  • Server
    • A server is a large, expensive computer designed to be the master of the network
    • The server is the core of the system that helps decides how data and resources are shared among the devices on the network
  • Router
    • Routers are used to join various networks together
    • This may be connecting a LAN to another LAN or a WAN
    • The router also acts as security between the different networks, only allowing authorised access
Software
  • Network Operating Software (NOS)
    • NOS is the software that is used to control the network
    • There is specific software for the servers and another software for the computers connected to the network
    • NOS prioritises and controls the devices on the network
    • NOS managers the security so that only authorised users can access the resources
  • Application software
    • There is also software that is designed to be used by computers within a network. These include:
      • Electronic mail
      • File transfer protocols
      • Web browser
      • Video conferencing
      • Instant messaging
Wired
  • Unshielded Twisted Pair cable (UTP)
    • Four pairs of twisted wires are twisted together as a group
    • Prevents outside interference because wires are not running parallel to any outside cables
    • Category 6 (CAT6) is currently the most popular for new installations
  • Fibre optic
    • Made from special glass or plastic strands
    • Transmit data through light pulses
    • Can carry data up to two kilometres as the light is not susceptible to electromagnetic interference
    • Two strands are required to for two way traffic as each strand only transmits data one way
Wireless
  • Radio waves
    • A transmitter is required to broadcast the signal and a receiver is required to accept it
    • This can be used over long distance or a short distance
    • Can be susceptible to noise
    • Bluetooth uses radio waves
  • Microwaves
    • Requires a concave dish, antenna and transceiver
    • Usually located on hills or towers as they require no interference between the sending and receiving dish
    • Can carry data up to two kilometres as the light is not susceptible to electromagnetic interference
    • Can handle large amounts of data over short distance
  • Infrared
    • Uses technology that is similar to a television remote control
    • Good for short distances of about 5 metres but can be slow
    • Infrared is used by (very old) laptops and mobiles
Threats
  • Viruses
    • Infect a computer by negatively impacting the way it works
    • Presents itself by being attached to an executable program
    • Can take data from computer to elsewhere
  • Worms
    • A virus that replicates itself, taking up space on a computer
    • Sends copies of itself to other computers
    • Can present itself via emails
  • Trojan
    • Infects a computer changing the way it is connected or the way it is displayed
    • Presents itself by tricking users into downloading an application then installing itself on their system
  • Spyware
    • Monitors the users actions and secretly collects information about users without them knowing
    • Information that may collected include personal or banking information, as well as websites they visit
  • Hackers
    • An unauthorized outsider accesses the information system in order to obtain confidential information
    • Usually they intend to look at or alter data
    • Run hack.bat to become the batman
Security
  • Antivirus software
    • Checks for viruses when the computer starts up, executable files are run, when files are accessed or copied
    • If a virus is detected the antivirus software will automatically warn the user or remove the virus
  • Firewalls
    • Used to restrict access to a network from outsiders
    • Uses a combination of hardware and software that will only allow authorised users into the system
  • Passwords
    • Used by the NOS to identify authorised users of the network and grant access to appropriate data and resources
    • Used in conjunction with a username
    • To be most effective passwords should follow protocols such as:
      • Alphanumeric
      • 8 or more characters
      • Changed regularly
  • Biometrics
    • Uses a human characteristic (such as finger print or retinal patterns) to identify authorised users of the network
    • More reliable than password as it must be the actual user who is identified
  • Back ups
    • Regular copies of the data and information are made and stored offsite
    • If the network data is affected, the latest back up can be used to restore the data on the network
